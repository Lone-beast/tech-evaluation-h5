## 🎯 部署步骤（5分钟完成）

### 步骤 1：注册/登录 Gitee

1. **打开浏览器访问**：https://gitee.com/
2. 点击右上角"注册"或"登录"
3. 完成登录

---

### 步骤 2：创建新仓库

1. 点击右上角的 **"+"** 号
2. 选择 **"新建仓库"**
3. 填写仓库信息：
   ```
   仓库名称：evaluation-h5
   介绍：科技特长生评测H5应用
   是否公开：✅ 公开（必须勾选）
   使用 README 初始化：❌ 不勾选
   ```
4. 点击 **"创建"** 按钮

---

### 步骤 3：上传文件

#### 方法 A：使用 Git 命令（推荐）

在终端执行以下命令：

```bash
# 1. 进入项目目录
cd /Users/zengyaojie/work/TRAE/企微召回评测/h5-prototype

# 2. 初始化 Git
git init

# 3. 添加所有文件
git add .

# 4. 提交
git commit -m "首次部署 H5 评测应用"

# 5. 添加远程仓库（替换 YOUR_USERNAME）
#    YOUR_USERNAME 就是您的 Gitee 用户名
git remote add origin https://gitee.com/YOUR_USERNAME/evaluation-h5.git

# 6. 推送代码
git push -u origin master
```

**如果推送时要求输入用户名和密码**：
- 用户名：您的 Gitee 用户名
- 密码：您的 Gitee 密码

#### 方法 B：网页上传（更简单）

1. 在 Gitee 仓库页面，点击 **"文件"** 标签
2. 点击 **"上传文件"** 按钮
3. 选择以下文件上传：
   - `index.html`
   - `assets` 文件夹（整个文件夹）
4. 在底部提交说明填写：`首次部署`
5. 点击 **"提交"**

---

### 步骤 4：启用 Gitee Pages

1. 在仓库页面顶部，点击 **"服务"** 标签
2. 在左侧菜单找到 **"Gitee Pages"**
3. 点击 **"启动"** 按钮
4. 配置信息：
   - **部署分支**：选择 `master`
   - **部署目录**：选择根目录 `/`
5. 点击 **"启动"** 或 **"更新"**

**等待 1-2 分钟**，系统会自动部署。

---

## 💡 常见问题

### Q1: Gitee Pages 显示"404"？
**A**: 等待 2-3 分钟，Gitee 需要时间初始化

### Q2: 页面样式错乱？
**A**: 确保 `assets` 文件夹已上传

### Q3: 微信中无法打开？
**A**: Gitee Pages 在微信中应该可以正常打开

### Q4: 想要自定义域名？
**A**:
1. 在 Gitee Pages 设置中添加自定义域名
2. 在域名 DNS 添加 CNAME 记录指向 `您的用户名.gitee.io`
